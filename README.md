# phxreports.com
Website

Notes and instructions on how to make changes to some elements of the website.

(1) To change images on any given page. Login to BlueHost. Go to leftside menu select Websites-choose site to edit-Wordpress Admin. This opens a new page. Go to leftside menu choose Pages-All Pages. 
(2) Scroll through pages until you locate the page you want to change. Select the page. Choose Edit with Elementor.
(3) This opens the page in edit mode. Choose image to change by selecting image, edit icon (pencil). Image appears in left side menu. Hover over it and click on choose image. This takes you to a new screen where you can upload files. Upload a new image. Once image has been uploaded, you need to select it. This will replace the old image with the newly uploaded one. 
(4) Go back to the page and change the caption to credit photographer.
(5) Go to the eyeball icon in top menu bar to preview the new page. If you are happy with the changes. Select Publish. If not go back and continue editing.
